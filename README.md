# At-A-Glance

A water-themed, responsive React website for a wastewater treatment solutions company.

## Features

- React + Vite
- React Router v6
- React Hook Form contact form
- GitHub Pages deployment support
- Custom water-inspired design system

## Setup

```bash
npm install
npm run dev
```

## Build & Deploy

```bash
npm run build
npm run preview
npm run deploy
```

## Notes

- The contact form sends data to Formspree. Replace the placeholder `FORM_ENDPOINT` in `src/pages/Contact.jsx` with your actual Formspree form ID.
- The app uses `vite.config.js` base path `/at-a-glance/` for GitHub Pages deployment.
